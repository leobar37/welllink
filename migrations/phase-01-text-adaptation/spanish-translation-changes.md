## Implementation Notes

### Regional Considerations

- **Mexico**: Use "consulta médica" instead of "cita médica" in some regions
- **Spain**: Use "médico/a" consistently
- **Latin America**: Adapt formal/informal tone based on target audience
- **Argentina**: Consider "doctor/a" vs "médico/a" preferences

### Medical Terminology Consistency

- Use "paciente" consistently (not "cliente" or "usuario")
- Use "consulta" or "cita médica" consistently
- Use "tratamiento" instead of "servicio" when appropriate
- Use "historia clínica" instead of "historia médica" in some regions

### Cultural Adaptations

- Adapt formality level to medical professional context
- Use appropriate medical terminology for target region
- Consider local healthcare system terminology
- Adapt appointment scheduling conventions
